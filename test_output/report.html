4. Detailed Findings

4.1 SQL Injection in Login Endpoint

IDVULN-0001
SeverityCritical
CVSS Score9.8 (AV:N/AC:L/PR:N/UI:N/S:U/C:H/I:H/A:H)
Affected Assethttps://example.com — /api/login
CWECWE-89
What This Means: User input is passed directly to a SQL query without sanitization.
Technical Description
The /api/login endpoint concatenates the username parameter directly into a SQL query string.
Code Locations
src/db/queries.ts:42-45
Unsanitized input used in SQL query
const query = `SELECT * FROM users WHERE name = ${name}`;
- const query = `SELECT * FROM users WHERE name = ${name}`;+ const query = 'SELECT * FROM users WHERE name = $1';+ const result = await db.query(query, [name]);
Evidence
Send a single quote in the username field to trigger a SQL error.
curl -X POST https://example.com/api/login -d "user=admin'--&pass=x"
Impact
An attacker can extract the entire database, modify data, or escalate privileges.
Remediation Steps
Use parameterized queries instead of string concatenation.

4.2 Cross-Site Scripting (XSS) in Search

IDVULN-0002
SeverityHigh
CVSS Score7.1 (AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N)
Affected Assethttps://example.com — /search
CWECWE-79
What This Means: Reflected XSS in the search parameter allows script injection.
Technical Description
The search query parameter is reflected in the response without encoding.
Evidence
Inject a script tag via the search parameter.
curl "https://example.com/search?q=<script>alert(1)</script>"
Impact
Session hijacking, credential theft, defacement.
Remediation Steps
HTML-encode all user input before rendering in the page.

4.3 Missing Rate Limiting on Password Reset

IDVULN-0003
SeverityMedium
CVSS Score5.3
Affected Assethttps://example.com — /api/reset
What This Means: The password reset endpoint has no rate limiting.
Technical Description
No rate limit headers observed; 1000 requests in 10 seconds all returned 200.
Evidence
Send rapid password reset requests.
for i in range(1000):
    requests.post('https://example.com/api/reset', json={'email': 'victim@test.com'})
Impact
Brute-force attacks on reset tokens or email flooding.
Remediation Steps
Add rate limiting (e.g., 5 requests per minute per IP).
